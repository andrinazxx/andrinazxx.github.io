"Unmute Your Arcade Game" BGM in Pure Data[2022]

Keywords:

Pure Data, Design, UIUX, Computer Music, Random Variables

Introduction

The “_Final_MUS171.pd” is my main PD(Pure Data) patch. The drum machine and the wind instruments built up the whole arcade machine background music generator. I imported an array with the layback drum beats sequence for my drum Kick, Snare, and Hats. For my wind instruments, I used 3 sets of random number generators to create random melody.

Also, I added some colors to my horizontal and vertical radio to create some vibrance.
